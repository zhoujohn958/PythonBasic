## 🤖 Cursor Agent 配置

### .cursorrules 文件说明
这个文件配置了Cursor AI Agent的行为规则：

#### 核心规则
- **文件创建控制**：不会自动创建新文件，除非用户明确要求
- **代码修改优先**：优先在现有文件中进行编辑
- **中文回复**：所有回复都使用中文简体
- **项目特定规则**：针对Python学习项目的特殊配置

#### 工作流程
1. 首先检查现有文件是否满足需求
2. 如果需要在现有文件中修改，直接进行编辑
3. 如果需要创建新文件，先询问用户
4. 确保所有修改都符合学习目标
5. 在每次修改后提供清晰的说明

#### 禁止行为
- 不自动创建配置文件（除非明确要求）
- 不创建日志文件或临时文件
- 不创建备份文件
- 不创建重复功能的文件

#### 允许行为
- 修改现有文件内容
- 在用户明确要求时创建新文件
- 提供代码示例和解释
- 帮助解决编程问题
